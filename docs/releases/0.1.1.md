# TypeCarrier 0.1.1

[English](0.1.1.en.md)

发布日期：2026-06-04

## 定位

0.1.1 是 TypeCarrier 的第二个小范围测试版本，用来稳定 iPhone / Android 到 Mac 的本地输入闭环。

这个版本适合自己试用、朋友试用或 GitHub 公开技术预览；不应被描述为成熟正式版。

GitHub Release 会提供完整源码、macOS development 测试包，以及 Android debug APK 侧载测试包。iOS 端只提供源码，不提供 GitHub 可安装包。

## 运行要求

- iOS 26.0 或更新版本。
- macOS 26.0 或更新版本。
- iPhone 与 Mac 位于同一个局域网内；推荐先用 iPhone 热点场景验证。
- Android 8.0 或更新版本可侧载 Android APK；Android 端仍处于 sender 测试阶段。
- Mac 端需要授权辅助功能权限，才能把收到的文本插入当前输入焦点。

## 本版包含

- iPhone 输入和发送界面。
- Android 输入和发送界面，支持手动连接、配对码和已配对 Mac 的快速重连。
- macOS 菜单栏接收端和主窗口入口。
- 基于 Multipeer Connectivity 的 iPhone -> Mac 局域网发现和传输。
- 基于局域网 TCP 的 Android -> Mac 发送通道。
- Mac 端收到文本后写入本地历史，并按当前配置执行粘贴。
- 连接状态、接收状态、粘贴诊断和失败解释。

## 0.1.1 变化

- 稳定 Android sender 的配对、手动连接、快速重连和发送链路。
- 稳定 Mac receiver 的 Android bridge、粘贴回执和接收历史记录。
- 改进 macOS 主窗口打开行为、接收记录工具栏和粘贴诊断。
- 增加 GitHub release workflow 的 Android debug APK 构建和上传。
- 将三端对外版本号统一为 0.1.1。

## 已知限制

- Android APK 是 debug 构建，仅用于侧载测试；还没有 release 签名、Play 分发或商店上架能力。
- iOS 端不提供 GitHub 可安装包；TestFlight 或 App Store 分发需要 Apple Developer Program。
- macOS zip 使用 Apple Development / Personal Team 签名，不是 Developer ID notarized 正式包，Gatekeeper 可能拦截。
- 不包含云同步、互联网中转、Windows 或 AI 转写能力。
- 多设备并发、跨 iPhone / Android sender 优先级和队列策略仍是后续能力。
- 自动粘贴依赖 macOS 当前焦点和辅助功能权限，失败时应优先检查 Mac 端状态和诊断日志。

## 发布前验证

发布前至少运行：

```sh
script/ci_sanity.sh
xcodebuild -project TypeCarrier.xcodeproj -scheme TypeCarrierCore -destination 'platform=macOS' test
xcodebuild -project TypeCarrier.xcodeproj -scheme TypeCarrieriOS -destination 'generic/platform=iOS Simulator' build
xcodebuild -project TypeCarrier.xcodeproj -scheme TypeCarrierMac -destination 'platform=macOS' build
cd Apps/Android && ./gradlew testDebugUnitTest assembleDebug
```

建议额外做一次真机验证：

1. 启动 Mac 端 TypeCarrier，并确认辅助功能权限已授权。
2. 启动 iPhone 端 TypeCarrier。
3. 在 iPhone 输入中文、英文和换行文本。
4. 将 Mac 光标放到 TextEdit、浏览器输入框或聊天输入框。
5. 从 iPhone 发送，确认文本出现在 Mac 当前光标处。
6. 侧载 Android APK 后，使用同一台 Mac 的配对码完成首次配对。
7. 从 Android 输入或语音转文字并发送，确认文本出现在 Mac 当前光标处。
8. 重启 Android 应用后再次发送，确认已配对 Mac 能快速重连且不需要重新输入配对码。
